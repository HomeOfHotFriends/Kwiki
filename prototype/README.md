# Performance-to-Typography Prototype

## How to Run with shttps

1. Make sure you have Node.js installed on your device.
2. Open a terminal and navigate to this `prototype` directory.
3. Run:

   npx shttps

4. This will start a secure local server. It will print a URL like `https://localhost:8080` or similar.
5. Open that URL in your browser (on the same device, or use your device's IP address if accessing from another device on the same network).

## Why use shttps?
- Browsers require HTTPS for microphone access.
- shttps is a simple way to serve static files over HTTPS.

## Troubleshooting
- If you get a security warning, you may need to accept the self-signed certificate in your browser.
- If you need help, let me know your device and OS for more specific instructions.
