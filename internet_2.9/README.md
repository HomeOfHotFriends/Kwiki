# Internet 2.9: Alien Simulated Internet

Welcome to Internet 2.9—a recursive, explorable folder-database game simulating an alien internet. Every file and folder in this repository is a node, artifact, or site in this network. Navigate, discover, and remix the knowledge, stories, and systems within.

## How to Explore
- Each subfolder is a "site" or "node" in the alien network.
- Index files provide alien metadata, navigation, and hints.
- Files are linked, referenced, or transformed to create a living, recursive web.

## Structure
- `/atomic_systems` — Alien logic, systems, and world rules
- `/rust-web-apps` — Digital artifacts and applications
- `/scripts` — Automation, mutation, and transformation engines
- `/sources` — Source texts and deep archives
- `/wiki` — Knowledge, history, and lore

## Begin your journey by exploring the folders and following the alien indexes. Remix, mutate, and expand as you go!
